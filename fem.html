FPT Event Management

Java Web application

My group project, a preparation before the incoming OJT
An application for FPT Event Event organizers to post their event on and everyone else can view, interact with them

Roles and Main features

Event Organizer

Create and update event

Students

Interact with events (view, follow, discuss)

Lecturers

Be invited to events as the supporter

Admin

Manage the app to make sure FPT society behaves properly

Simple working flow

First, event organizers will create and post their events to News feed. Then, when events are posted there, other users like students and lecturers can view and interact with those events.

Technology

Back-end

Server core JSP/Servlet
Database SQL Server (hosted on Digital Ocean)
Realtime Database Firebase Realtime database
Local database Room library & SQL-Lite
Architecture MVC

Front-end
