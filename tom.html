Android application

My two years side-project, a mobile application for managing flashcard, playing with them for learning any languages
Besides, I create a English-Vietnamese dictionary for it with over 10000 of words

Features

Create flashcard (can have both text and image)

Manage flashcard, group them into sets

Playing flashcard with four playing modes:

-Basic

-Puzzle

-Writing

-Quiz

English - Vietnamese dictionary

Translate with Google (unavailable now)

Create flashcard from translation

Floating dictionary as Bubble chat of Messenger app

Change UI theme

Technology

Framework Android
Used language Kotlin
Local database Room library & SQL-Lite
Architecture MVVM
DI library Dagger

My role

I was not lonely when starting this project. I suggested this idea to my friend and we worked together to build this app at the early stage. But then my friend did not feel interested in this project anymore so he decided to leave.

So I am responsible for building 95% of this app, with a little hope that I can publish it on Google Play Store (it failed). It takes 2 years because the first version of this app is very simple. There were only 3 buttons on home screen. Actually, I researched and referenced to another app (like TFlat) for improving my app functions and I did refactor this project idea and UI up to 5 times to come up with this final version. These time really help me increase my design skill.

As I said, I did almost everything in this project. Code, UI and Database Design
